## Filter 

The following categories can be selected as a filter for the ad-hoc search:
- **countries**
- **institutions**
- **authors**
- **journals**

<p align="center">
  <img  width=60% src="../../doc/select_countries_filter.gif">
</p>

After selecting a new filter from the previous categories the previous search results are updated automatically. The selected filters can also be deselected by clicking the delete button of the unwanted filter.


>Note: The displayed map provides a general overview of the global distribution of the publications and is therefore not updated based on selected filters.
